# Adding the Material Plugin to Lumerical

Download `.dll` files from [the Notebooks with MyST Markdown documentation](https://jupyterbook.org/file-types/myst-notebooks.html). Then copy `.dll` file to the following address:
```
?:\???\Lumerical\v###\bin\plugins\materials\File_Name.dll
```

```{seealso}
For Example:

C:\Program Files\Lumerical\v212\bin\plugins\materials
```
